# AGENTS Instructions for This Repository

This is a beginner-friendly static HTML/CSS/JavaScript project.

## Core Working Principles
- Do not add third-party libraries unless explicitly requested.
- Keep code simple and readable.
- Prefer small, focused changes.
- For bug fixes, make the smallest safe change.

## UI and UX Boundaries
- Do not redesign the UI unless the task asks for it.

## localStorage Safety Rules
- Do not change the localStorage key unless explicitly requested.
- Do not clear or migrate existing localStorage data unless explicitly requested.

## Testing and Verification Notes
- After changing JavaScript, explain how to manually test the change.
- For new features, include normal cases, edge cases, and persistence checks in the testing notes.

## Documentation Scope
- Keep README.md updated only when documentation changes are part of the task.
